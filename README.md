# Sales System

This is a **simple Python Sales System** project built with object-oriented programming.  
It uses JSON for data storage and the **Faker** library to generate realistic fake sales entries.

---

## 📌 Features

✔ Generate fake sales entries  
✔ Add a new sale  
✔ List all sales with indexes  
✔ Update a field of a sale record  
✔ Delete a sale by index  
✔ Search records by specific field  
✔ Simple CLI menu (switch/case)

---

## 🛠️ Technologies

- Python 3.10+  
- Faker library  
- JSON for simple storage  
- Object-oriented principles  
- Enum for safe field selection
